# Лабораторная работа: Обработка текстов с помощью нейронных сетей

## Цель работы
Изучить методы обработки текста с использованием нейронных сетей, освоить токенизацию, создание последовательностей, применение Embedding слоёв и полносвязных архитектур для задач классификации текстов.

---

## Задание 1 (Lite): Классификация писателей

**Требование:**
Создать 9 нейросетей с различными гиперпараметрами:
- Размер словаря: 15000
- Ширина окна (WIN_SIZE): 1000, 1500, 2000
- Шаг (WIN_HOP): 100, 300, 500

**Что сделано:**
- Загружен датасет с текстами 6 русских писателей
- Проведена токенизация и создание последовательностей
- Реализована модель: Embedding → Flatten → Dense(256) → Dropout(0.3) → BatchNormalization → Dense(128) → Dropout(0.3) → Softmax
- Проведено 9 экспериментов, выбраны оптимальные параметры

**Достигнутая точность:** 82.48% (WIN_SIZE=2000, WIN_HOP=100)

---

## Задание 2 (Pro): Классификация отзывов о Tesla

**Требование:**
- Загрузить базу отзывов о Tesla
- Сбалансировать выборку (количество примеров каждого класса — одного порядка)
- Добиться точности 85–90% на проверочной выборке

**Что сделано:**
- Загружены позитивные и негативные отзывы
- Выполнена балансировка: разбиение текстов на фрагменты (500 символов, шаг 250)
- Исключена утечка данных (разделение на train/test до разбиения на фрагменты)
- Реализована модель Bag of Words: CountVectorizer (max_features=5000, ngram_range=(1,2))
- Архитектура: Dense(128) → Dropout(0.5) → BatchNorm → Dense(64) → Dropout(0.5) → Dense(32) → Dropout(0.4) → Sigmoid

**Достигнутая точность:** 84.73%

## Задание 3 (Ultra Pro): Классификация 20 русских писателей

**Требование:**
- Загрузить базу из 20 русских писателей
- Создать обучающую и проверочную выборки с балансировкой (использовать цикл)
- Проверочная выборка — 20% от общей
- Bag of Words модель: точность не менее 95%
- Embedding модель: точность 75–80%

**Что сделано:**
- Загружены тексты 20 писателей
- Балансировка: разбиение на фрагменты (1000 символов, шаг 500) с использованием цикла
- Bag of Words: CountVectorizer (max_features=4000, ngram_range=(1,2))
- Embedding: MAX_WORDS=15000, MAX_LEN=200, архитектура: Embedding(64) → Flatten → Dense(128) → Dropout(0.4) → BatchNorm → Dense(64) → Dropout(0.3) → Softmax

**Достигнутая точность:**
- Bag of Words: 91.72% 
- Embedding: 88.10%
