# Лабораторная работа 1: Обработка текстов с помощью нейронных сетей

## Цель работы
Изучить методы обработки текста с использованием нейронных сетей, освоить токенизацию, создание последовательностей, применение Embedding слоёв и полносвязных архитектур для задач классификации текстов.

## Задачи работы

### Задание 1 (Lite): Классификация писателей
Реализовать и обучить нейронную сеть для распознавания 6 русских писателей по фрагментам их произведений.

**Выполнено:**
- Загрузка и подготовка датасета `writers.zip`
- Токенизация текстов с параметрами: `VOCAB_SIZE = 15000`
- Создание 9 экспериментальных моделей с различными гиперпараметрами:
  - `WIN_SIZE`: 1000, 1500, 2000
  - `WIN_HOP`: 100, 300, 500
- Архитектура модели: `Embedding(128) → Flatten → Dense(256) → Dropout(0.3) → BatchNorm → Dense(128) → Dropout(0.3) → Softmax`
- Сравнение результатов и выбор оптимальных параметров

**Результаты экспериментов:**
| Эксперимент | WIN_SIZE | WIN_HOP | Лучшая val_accuracy |
|-------------|----------|---------|---------------------|
| 1 | 1000 | 100 | 0.7975 |
| 2 | 1000 | 300 | 0.6602 |
| 3 | 1000 | 500 | 0.4649 |
| 4 | 1500 | 100 | 0.8131 |
| 5 | 1500 | 300 | 0.7306 |
| 6 | 1500 | 500 | 0.5435 |
| 7 | 2000 | 100 | **0.8248** |
| 8 | 2000 | 300 | 0.7607 |
| 9 | 2000 | 500 | 0.5791 |

**Лучший результат:** 82.48% (WIN_SIZE=2000, WIN_HOP=100)

### Задание 2: Классификация отзывов о Tesla
Реализовать бинарную классификацию отзывов (позитивные / негативные) с балансировкой выборки и целевой точностью 85-90%.

**Выполнено:**
- Загрузка датасета `tesla.zip` (негативные и позитивные отзывы)
- Балансировка выборки: разбиение текстов на фрагменты (CHUNK_SIZE=500, STEP=250)
- Разделение на обучающую (80%) и тестовую (20%) выборки без утечки данных
- Векторизация Bag of Words (CountVectorizer)
- Архитектура модели: `Dense(128) → Dropout(0.5) → BatchNorm → Dense(64) → Dropout(0.5) → Dense(32) → Dropout(0.4) → Sigmoid`
- Использование class_weight для балансировки классов

**Результат:** Точность на тестовой выборке: **~85%** (цель достигнута)

### Задание 3 (Ultra Pro): Классификация 20 русских писателей
Реализовать классификацию 20 русских писателей с использованием двух подходов: Bag of Words и Embedding.

**Выполнено:**
- Загрузка датасета `20writers.zip`
- Балансировка выборки: разбиение на фрагменты (CHUNK_SIZE=1000, STEP=500) с использованием цикла
- Разделение на обучающую (80%) и тестовую (20%) выборки

**Модель 1 - Bag of Words:**
- Параметры: `max_features=4000`, `ngram_range=(1,2)`, `min_df=2`, `max_df=0.8`
- Архитектура: `Dense(256) → Dropout(0.4) → BatchNorm → Dense(128) → Dropout(0.4) → BatchNorm → Dense(64) → Dropout(0.3) → Softmax`
- Точность: **91.72%** (ограничение памяти Colab, целевые 95% не достигнуты)

**Модель 2 - Embedding:**
- Параметры: `MAX_WORDS=15000`, `MAX_LEN=200`
- Архитектура: `Embedding(64) → Flatten → Dense(128) → Dropout(0.4) → BatchNorm → Dense(64) → Dropout(0.3) → Softmax`
- Точность: **88.10%** (целевой диапазон 75-80% выполнен)

**Итоговая таблица:**
| Модель | Точность | Требование | Статус |
|--------|----------|------------|--------|
| Bag of Words | 91.72% | ≥95% | ⚠️ (ограничение памяти) |
| Embedding | 88.10% | 75–80% | ✅ |
