# Лабораторная работа: Обработка текстов с помощью нейронных сетей

## Цель работы
Изучить методы обработки текста с использованием нейронных сетей, освоить токенизацию, создание последовательностей, применение Embedding слоёв и полносвязных архитектур для задач классификации текстов.

## Задачи работы

1. **Задание Lite:** Реализовать и обучить нейронную сеть для классификации 6 русских писателей. Провести 9 экспериментов с различными гиперпараметрами (размер окна и шаг). Добиться максимальной точности на проверочной выборке.

2. **Задание Pro:** Реализовать и обучить нейронную сеть для бинарной классификации отзывов об автомобиле Tesla (позитивные / негативные). Обеспечить балансировку выборки. Добиться точности 85–90% на проверочной выборке.

3. **Задание Ultra Pro:** Реализовать и обучить нейронную сеть для классификации 20 русских писателей, используя два подхода:
   - Модель Bag of Words (целевая точность ≥95%)
   - Embedding-модель (целевая точность 75–80%)
   Обеспечить балансировку выборки с использованием цикла, проверочная выборка — 20%.
